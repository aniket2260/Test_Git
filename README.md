### Key Git Commands for the README Workflow

The following commands are useful when managing your `README.md` file:

*   **`touch README.md`**: Creates a new `README.md` file in the current directory (Unix/Linux/macOS).
*   **`echo "Project info" >> README.md`**: Adds initial content to the `README.md` file from the command line.
*   **`git status`**: Shows the new or modified files that haven't yet been committed.
*   **`git add README.md`**: Stages the `README.md` file for the next commit.
*   **`git commit -m "Add README"`**: Records the file changes to the local repository.
